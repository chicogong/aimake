> 更新日期: 2026-01-09状态: 规划中

## 产品定位

**AIMake** - AI 语音内容生成平台

一句话描述：**用 AI 快速生成播客、有声书、配音、教育内容**

---

## 核心价值

| 传统方式     | AIMake        |
| ------------ | ------------- |
| 需要录音设备 | 只需文字输入  |
| 需要配音演员 | AI 多音色可选 |
| 需要后期剪辑 | 一键生成成品  |
| 耗时几小时   | 几秒出结果    |

### AI/TTS

| 供应商               | 免费额度        | 付费价格        | 优缺点             | 推荐场景       |
| -------------------- | --------------- | --------------- | ------------------ | -------------- |
| **腾讯云 TTS**       | 800万字符/3个月 | ¥2/万字符       | 中文音质好，成本低 | **MVP 主力**   |
| **Google Cloud TTS** | 400万字符/月    | $0.004/1K chars | 免费额度持续，稳定 | **主备份**     |
| **Azure TTS**        | 500万字符/月    | $0.001/1K chars | **最便宜**，多语言 | **付费兜底**   |
| **OpenAI TTS**       | 无              | $0.015/1K chars | 音质优秀，简单     | Pro 用户高质量 |
| **MiniMax**          | 1万字符/月      | $0.03/1K chars  | 中文超自然         | Pro 用户中文   |
| **ElevenLabs**       | 1万字符/月      | $0.30/1K chars  | 音质最佳，**最贵** | 企业用户       |

**多供应商策略**（详见 [tts-providers-comparison.md](./tts-providers-comparison.md)）:

- **Phase 1 (MVP)**: 腾讯云（800万字符免费）→ Google（备用）
- **Phase 2 (增长)**: Google（400万/月免费）+ Azure（超出付费）
- **Phase 3 (规模)**: 智能路由 - 免费用户用免费额度，Pro 用户用 OpenAI/MiniMax

**成本优化**:

- MVP 阶段：1000 免费用户 × 10分钟/月 = 20万字符 → **完全免费**（腾讯云额度可用 33 年）
- 增长阶段：10000 用户，Google + Azure 组合 → 月成本 **$7**
- 规模化：10000 用户（80%免费+20%Pro）→ 月成本 **$257**，收入 $19,800，利润率 **98.7%**

## 风险与挑战

| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |

| 音质不够好     | **智能路由**: 根据场景选择供应商<br>- 免费用户: 腾讯云/Google（音质 4.4-4.5 分）<br>- Pro 用户: OpenAI/MiniMax（音质 4.7+ 分）<br>- 支持用户自选音色                 | ✅ 已规划 |
| 供应商不稳定   | **多供应商冗余**: 主-备-兜底三层架构<br>- 主: 腾讯云 → 备: Google → 兜底: Azure<br>- 自动故障切换，99.9% 可用性                                                      | ✅ 已规划 |
