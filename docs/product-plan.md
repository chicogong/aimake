# AIMake 产品规划

> 更新日期: 2026-01-09
> 状态: 规划中

---

## 产品定位

**AIMake** - AI 语音内容生成平台

一句话描述：**用 AI 快速生成播客、有声书、配音、教育内容**

---

## 核心价值

| 传统方式 | AIMake |
|----------|--------|
| 需要录音设备 | 只需文字输入 |
| 需要配音演员 | AI 多音色可选 |
| 需要后期剪辑 | 一键生成成品 |
| 耗时几小时 | 几秒出结果 |
| 成本高昂 | 免费起步 |

## 核心功能

### MVP 功能 (Phase 1)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **文本转语音** | 输入文字，生成音频 | P0 |
| **多音色选择** | 10+ 种自然音色 | P0 |
| **音频下载** | MP3/WAV 导出 | P0 |
| **播放预览** | 在线试听生成结果 | P0 |
| **历史记录** | 保存已生成的音频 | P1 |

### 进阶功能 (Phase 2)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **双人对话生成** | 输入话题，生成播客对话 | P1 |
| **长文转有声书** | 文章/小说转音频，自动分章 | P1 |
| **情感/语速控制** | 调节语调、语速、情感 | P1 |
| **批量生成** | 上传文档，批量转换 | P2 |

### 高级功能 (Phase 3)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **语音克隆** | 用户上传样本，克隆音色 | P2 |
| **多语言** | 支持英语、日语等 | P2 |
| **背景音乐** | 自动添加 BGM | P2 |
| **API 接口** | 开发者接入 | P2 |

## 技术栈

### 前端

| 技术 | 用途 |
|------|------|
| React 18 | UI 框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Tailwind CSS | 样式 |
| Zustand | 状态管理 |

### 后端

| 技术 | 用途 |
|------|------|
| Python / FastAPI | API 服务 |
| PostgreSQL | 用户数据 |
| Redis | 缓存/队列 |
| S3/MinIO | 音频存储 |

### AI/TTS

| 选项 | 优缺点 |
|------|--------|
| **ElevenLabs API** | 音质最好，成本高 |
| **OpenAI TTS** | 简单易用，音色少 |
| **Azure TTS** | 企业级，多语言 |
| **CosyVoice (自建)** | 成本低，需要 GPU |

**MVP 建议**: 先用 ElevenLabs/OpenAI API，后续考虑自建

## 风险与挑战

| 风险 | 应对策略 |
|------|----------|
| TTS API 成本高 | 先用 API，验证需求后自建 |
| 音质不够好 | 选择顶级 API (ElevenLabs) |
| 用户获取难 | 内容营销 + 免费版引流 |
| 竞争激烈 | 聚焦特定场景，做深做透 |
